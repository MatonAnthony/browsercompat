The browsercompat API in his version 1 is going to be unsupported and removed in favor of version 2.

The API in version 1 is currently based on JSON API RC1.0 which is now 2 years old and has evolved to a full JSON API v1.0 which is served by our API version 2.

Links to both APIs:

  • Browser Compatibility API version 2 - [RECOMMENDED] The maintained and supported API.
  • Browser Compatibility API version 1 - [DEPRECATED] The API version 1 which is going away in the future.
